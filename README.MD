## InstalaciГіn y configuraciГіn

### 1. Clonar el repositorio

```bash

```

### 2. Crear y activar el entorno virtual


SabrГЎs que estГЎ activo porque tu terminal muestra `(venv)` al inicio de la lГӯnea.

### 3. Instalar dependencias

### 4. Configurar variables de entorno

> El archivo `.env` **no se sube a GitHub** (estГЎ en `.gitignore`), ya que contiene credenciales sensibles.

### 5. Crear la base de datos

Ejecuta el script `database.sql` en MySQL Workbench (o el cliente que uses) para crear la base de datos `stockpro` y sus tablas (`usuarios`, `categorias`, `productos`, `movimientos_stock`).

### 6. Generar la contraseГұa del usuario administrador

El usuario de prueba se crea con una contraseГұa temporal (no hasheada). Genera el hash real asГӯ:

Crea un archivo temporal `generar_hash.py` en la raГӯz del proyecto:

```python

```

EjecГәtalo:

```bash

```

Copia el valor impreso (algo como `$2b$12$abcdefg...`) y actualiza el usuario en MySQL:

```sql

SET password_hash = 'PEGA_AQUI_EL_HASH_GENERADO'

```

Una vez hecho esto, puedes borrar `generar_hash.py` вҖ” ya cumpliГі su funciГіn.
